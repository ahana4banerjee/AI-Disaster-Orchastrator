# Public Features

## Overview

The Public Portal is designed for citizens, students, travelers, and local communities.

Its purpose is to improve disaster awareness, preparedness, and risk understanding.

---

# Module 1: Personal Risk Checker

## Purpose

Allow users to understand disaster risks in their region.

## Inputs

* Country
* State/Region

## Outputs

* Most common disaster types
* Historical frequency
* Historical impact
* Regional risk score

---

# Module 2: Nearby Disaster Explorer

## Purpose

Explore historical disasters near the user's region.

## Features

Display:

* Recent disasters
* Disaster categories
* Historical trends
* Impact summaries

---

# Module 3: Disaster Awareness Hub

## Purpose

Educate users about disaster risks.

## Categories

* Floods
* Cyclones
* Earthquakes
* Landslides
* Droughts
* Heatwaves
* Tsunamis

For each:

* Description
* Warning signs
* Common impacts
* Historical examples

---

# Module 4: Preparedness Assistant

## Purpose

Generate preparedness recommendations.

## Inputs

* Region
* Disaster Type

## Outputs

Customized preparedness checklist.

Example:

Cyclone Preparation

* Emergency Kit
* Backup Power
* Water Storage
* Evacuation Planning

---

# Module 5: Family Emergency Planner

## Purpose

Create personalized emergency plans.

## Inputs

* Family Size
* Elderly Members
* Children
* Vehicles Available

## Outputs

* Emergency Supply List
* Evacuation Recommendations
* Emergency Contacts Checklist

---

# Module 6: Readiness Score

## Purpose

Measure preparedness level.

## Inputs

Preparedness questionnaire.

## Output

Readiness Score

0–100

### Categories

* Poor
* Fair
* Good
* Excellent

---

# Module 7: Historical Disaster Explorer

## Purpose

Allow users to explore EM-DAT records.

## Filters

* Country
* Year
* Disaster Type

## Outputs

Historical disaster cards and statistics.

---

# Module 8: Regional Disaster Insights

## Purpose

Provide simplified regional analytics.

## Display

* Most common disasters
* Highest impact events
* Average annual losses

---

# Module 9: Safety Recommendations

## Purpose

Provide actionable safety guidance.

## Example

For Floods

Before:

* Emergency Kit

During:

* Evacuate Low Areas

After:

* Avoid Contaminated Water

---

# Module 10: Public AI Assistant

## Purpose

Answer disaster preparedness questions.

## Example Questions

* How should I prepare for a cyclone?
* What should I do during an earthquake?
* How much water should my family store?

Provides educational and preparedness-focused responses.
